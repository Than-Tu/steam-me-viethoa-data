# Steam Mê Việt Hóa - Static Data

Repository này chứa dữ liệu tĩnh (JSON) cho extension Steam Mê Việt Hóa.
Dữ liệu được host miễn phí trên GitHub Pages để tối ưu chi phí và băng thông.

## Cấu trúc

- `/data/games.json`: Danh sách các game hỗ trợ Việt Hóa.
- `/data/translations.json`: Metadata chi tiết về các bản dịch.
- `/scripts/`: Các script utility để update data.

## Usage

Extension sẽ fetch data từ: `https://[USERNAME].github.io/steam-me-viethoa-data/data/games.json`
